# posDemux

The goal of posDemux is to provide tools for tools for handling reads
with combinatorial barcodes, and multiple adapter regions. This includes
utilities for demultiplexing, sequence segmenting and filtering. The
package is intended to work with single-cell RNA-seq data with multiple
barcoding (e.g. SPLiT-seq and PETRI-seq).

## Installation

You can install the development version of posDemux from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("yaccos/posDemux")
```
